# A Unique Conversation

A terminal visual novel. Two men, late coffee, thick fog, cold April.

*by Leib Weissman*

---

## Requirements

```bash
python3 -m pip install blessed
```

## Run

```bash
python3 game.py
```

From the project root:

```bash
python3 leib/a-unique-conversation/game.py
```

## Controls

- **Any key** — advance to the next scene
- **Ctrl+C** — exit

## Notes

- Best in a full-size terminal with 24-bit color support (iTerm2, Windows Terminal, most modern terminals)
- Do not resize the window mid-game
- 12 scenes. One direction. No going back.
